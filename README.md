# Hesap Makinesi
Ben bilgisayar proglamcılığı 2-A sınıfdan Günay Akdeniz projeyi html üzerinden yapıp.
Windows tabanlı bir bilgisayardan projryi test edip çalıştırdım.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Özellikler
- Toplama
- Çıkarma
- Çarpma
- Bölme
- Temizleme (C) butonu
- Butonlu kullanım

## Nasıl Çalıştırılır?
1. Proje klasörü indirilir.
2. `hesapmakinesi.html` dosyası çift tıklanarak herhangi bir web tarayıcısında açılır.
3. Butonlar kullanılarak işlemler yapılır.

## Amaç
Bu projenin amacı, HTML, CSS ve JavaScript kullanarak temel bir web uygulaması geliştirmeyi öğrenmektir.
